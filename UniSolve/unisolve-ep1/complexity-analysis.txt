N numbers summation algorithm analysis:
----------------------------------------
1. input n numbers 
2. sum n numbers
3. print sum

complexity O(N) or variable complexity

complexity notation:
1. Big O - Upper bound / at most i.e  O(1), O(n), O(n^2), O(n^3), O(sqrt(n)), O(log n) ...
2. Omega Ω - lower bound / at least
3. Theta θ - tight bound / exact
